---
name: telex-codec
description: >
  Encode text into Telex network audio WAV files and decode Telex recordings
  back into text, using ITA2 Baudot code at 50 baud with FSK modulation and
  telex-specific signaling conventions including answerback codes and message
  formatting. Use this skill whenever the user mentions telex, teleprinter
  network, TWX, TLX, telex machine, telex message, answerback, WRU signal,
  ITA2 50 baud, landline teleprinter, or wants to simulate telex network
  communications. Distinct from the RTTY skill which covers radio-based
  radioteletype — this skill covers the landline telex network with its
  specific conventions, routing, and signaling. Covers encoding (text to WAV)
  and decoding (WAV to text).
